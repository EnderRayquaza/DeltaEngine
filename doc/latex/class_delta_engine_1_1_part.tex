\doxysubsection{Detailed Description}
The \mbox{\hyperlink{namespace_delta_engine}{Delta\+Engine}} item base. It contains a sprite (with SFML) and an hitbox (with Box2D). 

Friend with \mbox{\hyperlink{class_delta_engine_1_1_game}{Game}}, \mbox{\hyperlink{class_delta_engine_1_1_object}{Object}} and \mbox{\hyperlink{class_delta_engine_1_1_entity}{Entity}} \mbox{\hyperlink{class_delta_engine_1_1_part}{Part}} are a combination of a texture (SFML) and an hitbox (Box2D). They can be animated and Shader can be applied on. \begin{DoxySeeAlso}{See also}

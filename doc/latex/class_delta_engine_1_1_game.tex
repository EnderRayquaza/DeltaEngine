\doxysubsection{Detailed Description}
A class that contains yours \mbox{\hyperlink{class_delta_engine_1_1_object}{Object}}, \mbox{\hyperlink{class_delta_engine_1_1_entity}{Entity}}, \mbox{\hyperlink{class_delta_engine_1_1_light}{Light}} and \mbox{\hyperlink{class_delta_engine_1_1_shader_manager}{Shader\+Manager}}. 

\mbox{\hyperlink{class_delta_engine_1_1_game}{Game}} contains infos about your project (name, version, debug/release mode, etc...) and your \mbox{\hyperlink{class_delta_engine_1_1_object}{Object}}, \mbox{\hyperlink{class_delta_engine_1_1_entity}{Entity}}, \mbox{\hyperlink{class_delta_engine_1_1_light}{Light}} and Shader (in a \mbox{\hyperlink{class_delta_engine_1_1_shader_manager}{Shader\+Manager}}). This class draws everything on the window too.
